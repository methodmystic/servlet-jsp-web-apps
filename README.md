### Servlet-jsp-web-apps
#### Concepts Covered

**🔧 Core Servlet Concepts**
- Servlet Lifecycle & `service()` method — Understanding how a servlet is initialized, handles requests, and is destroyed
- HTML Form Handling (GET/POST) — Capturing user input from HTML forms and processing it inside a servlet

**🗄️ Database (registration-web)**
- JDBC Database Connectivity — Connecting a Java web app to MySQL using `DriverManager` and `Connection`
- PreparedStatement — Safely inserting user data into the database, preventing SQL injection attacks

**🔀 Request Handling (request-dispatching)**
- RequestDispatcher (`forward`) — Forwarding a request from one servlet to another without a browser redirect
- HttpSession Management — Storing and retrieving user data across multiple servlets using session attributes

**📄 JSP (JspWeb)**
- JSP Scripting Elements — Using declarations (`<%! %>`), scriptlets (`<% %>`), and expressions (`<%= %>`) inside JSP pages
- Dynamic Response Generation — Generating personalized HTML output on the server side using Java logic inside JSP
